# 灵光收集箱 Preview

这是手机优先的轻量 HTML 预览，用来验证“快速记录 + 回头处理”的核心手感。

当前方向已经收敛为：

- 只有两个主界面：`记` 和 `看`。
- 使用深色 UI，首页不显示应用大标题，只保留日期和输入区。
- 首页只负责快速输入，不再选择想法/待做。
- 所有内容都是同一种记录，不再区分类型，也不再挂到主题下面。
- 首页保存按钮放在屏幕中线附近，下方最多露出 3 条进行中记录，更多内容去 `看` 页面。
- 首页的进行中记录带圆环快捷操作，点圆环直接完成，也可以编辑内容。
- 查看页按时间线展示记录，可以搜索正文，并筛选全部、未处理、进行中、已完成、已放弃。
- 查看页里的每条记录内容都可以编辑。
- 圆环是主状态推进入口：未处理 -> 进行中 -> 已完成；已完成或已放弃点圆环会重新打开。
- `全部` 默认不混入已放弃内容；已放弃有单独筛选入口。
- 每条记录自动保存创建时间；开始时记录开始时间；完成或放弃时记录对应时间。
- 不做主题、子待办、搜索、标签、垃圾箱、导出和系统提醒。

`preview-b.html` 保留为更早的时间线草案；主预览入口是 `index.html`。
